To ensure you are ready for the course, please complete these essential setup steps:

### 1. Kaggle Account
You will need a Kaggle account to access the course's codelabs.
* **Sign up:** [Create a Kaggle account](https://www.kaggle.com)
* **Learn Notebooks:** [Get familiar with how Kaggle Notebooks work](https://www.kaggle.com/docs/notebooks)
* **Phone Verify:** **This is mandatory for the codelabs.** [Verify your account via phone](https://www.kaggle.com/account/phone-verification) to enable internet access in your notebooks.

### 2. AI Studio Account
The codelabs use the Gemini API, which requires an API key from AI Studio.
* **Sign up:** [Create an AI Studio account](https://aistudio.google.com/)
* **Get Key:** Ensure you can [generate an API key](https://aistudio.google.com/app/apikey). You will add this as a secret in your Kaggle notebooks.

### 3. Kaggle Discord
This is the community hub for connecting with fellow learners and asking questions.
* **Sign up:** [Create a Discord account](https://discord.com/register)
* **Join Server:** [Join the Kaggle Discord server](https://discord.gg/kaggle)
